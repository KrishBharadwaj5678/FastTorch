<h1 align="center">🔦 Fast Torch</h1>

<p align="center">
  Fast Torch は Flutter を使用して開発された軽量な懐中電灯アプリです。ワンタップでデバイスのライトをすぐにオン・オフでき、暗い場所で素早く利用できます。
</p>

![FastTorchDemo](https://github.com/KrishBharadwaj5678/FastTorch/blob/main/FastTorchDemo.jpeg)

## <img src="https://github.com/KrishBharadwaj5678/FastTorch/raw/main/assets/icon/readme/features.gif" width="35"/> 主な機能

| 機能 | 説明 |
|------|------|
| 🔦 **瞬時にライトを切り替え** | ワンタップでデバイスの懐中電灯をオン・オフできます。 |
| 📳 **バイブレーションフィードバック** | ライトの切り替え時に、オプションでバイブレーションによるフィードバックを提供します。 |
| 🚀 **軽量・高速** | 高速動作と最小限のアプリサイズを実現するよう最適化されています。 |
| 🌑 **ダークモード対応** | 洗練されたダークテーマのユーザーインターフェースを採用しています。 |

## <img src="https://github.com/KrishBharadwaj5678/FastTorch/raw/main/assets/icon/readme/gettingStarted.gif" width="35"/> はじめに

### 1️⃣ リポジトリをクローンする

```bash
git clone https://github.com/KrishBharadwaj5678/FastTorch.git
```

### 2️⃣ プロジェクトディレクトリへ移動する

```bash
cd FastTorch
```

### 3️⃣ 依存関係をインストールする

```bash
flutter pub get
```

### 4️⃣ アプリを実行する

```bash
flutter run
```
