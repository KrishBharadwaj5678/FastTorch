<h1 align="center">🔦 Fast Torch</h1>

<p align="center">
  Fast Torch é um aplicativo de lanterna leve desenvolvido com Flutter. Ele permite ligar e desligar instantaneamente a lanterna do seu dispositivo, sendo perfeito para ambientes escuros.
</p>

![FastTorchDemo](https://github.com/KrishBharadwaj5678/FastTorch/blob/main/FastTorchDemo.jpeg)

## <img src="https://github.com/KrishBharadwaj5678/FastTorch/raw/main/assets/icon/readme/features.gif" width="35"/> Recursos

| Recurso | Descrição |
|---------|-----------|
| 🔦 **Lanterna Instantânea** | Ligue ou desligue a lanterna do dispositivo com apenas um toque. |
| 📳 **Feedback por Vibração** | Vibração opcional ao ativar ou desativar a lanterna. |
| 🚀 **Leve e Rápido** | Otimizado para oferecer alto desempenho e um tamanho reduzido. |
| 🌑 **Compatível com Modo Escuro** | Interface elegante com tema escuro. |

## <img src="https://github.com/KrishBharadwaj5678/FastTorch/raw/main/assets/icon/readme/gettingStarted.gif" width="35"/> Como Começar

### 1️⃣ Clone o repositório

```bash
git clone https://github.com/KrishBharadwaj5678/FastTorch.git
```

### 2️⃣ Acesse o diretório do projeto

```bash
cd FastTorch
```

### 3️⃣ Instale as dependências

```bash
flutter pub get
```

### 4️⃣ Execute o aplicativo

```bash
flutter run
```
