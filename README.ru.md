<h1 align="center">🔦 Fast Torch</h1>

<p align="center">
  Fast Torch это лёгкое приложение-фонарик, разработанное с использованием Flutter. Оно позволяет мгновенно включать и выключать фонарик устройства, что особенно удобно в тёмных местах.
</p>

![FastTorchDemo](https://github.com/KrishBharadwaj5678/FastTorch/blob/main/FastTorchDemo.jpeg)

## <img src="https://github.com/KrishBharadwaj5678/FastTorch/raw/main/assets/icon/readme/features.gif" width="35"/> Возможности

| Возможность | Описание |
|-------------|----------|
| 🔦 **Мгновенное включение фонарика** | Включение и выключение фонарика одним касанием. |
| 📳 **Виброотклик** | Необязательная вибрация при переключении фонарика. |
| 🚀 **Лёгкое и быстрое приложение** | Оптимизировано для высокой скорости работы и минимального размера. |
| 🌑 **Поддержка тёмной темы** | Современный интерфейс, идеально подходящий для тёмного режима. |

## <img src="https://github.com/KrishBharadwaj5678/FastTorch/raw/main/assets/icon/readme/gettingStarted.gif" width="35"/> Начало работы

### 1️⃣ Клонируйте репозиторий

```bash
git clone https://github.com/KrishBharadwaj5678/FastTorch.git
````

### 2️⃣ Перейдите в директорию проекта

```bash
cd FastTorch
```

### 3️⃣ Установите зависимости

```bash
flutter pub get
```

### 4️⃣ Запустите приложение

```bash
flutter run
```
